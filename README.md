# 卫戍协议：盟约 - 自走棋塔防游戏

## 游戏特色

### 核心机制
- **塔防+自走棋融合**：结合明日方舟塔防机制与自走棋玩法
- **地图系统**：9x4紧凑地图，包含障碍物、入侵点和保护点
- **干员合成**：三个同名同精英度干员自动合成为更高精英度
- **商店升级**：提升商店等级获得更强干员
- **回合制金币**：每回合金币不继承，增加策略性

### 地图元素
- 🚪 **敌方入侵点**：右上角和右下角（红色脉冲）
- 🏠 **我方保护点**：左下角（绿色脉冲）
- 🗿 **障碍物**：敌人无法穿过，干员无法部署
- 敌人按最短路径自动寻路移动

### 干员系统
- **5个星级**：⭐~⭐⭐⭐⭐⭐
- **3个精英度**：普通、精一★、精二★★
- **6种职业**：
  - ⚔️ 近卫：高攻击，阻挡1
  - 🏹 狙击：远程物理，阻挡0
  - 🛡️ 重装：高防御，阻挡3
  - 💊 医疗：治疗友军，阻挡0
  - ✨ 术师：法术伤害，阻挡0
  - 🎖️ 先锋：平衡属性，阻挡2

**阻挡机制说明**：
- 干员会显示当前阻挡数/最大阻挡数（如 2/3）
- 当阻挡的敌人数量达到上限时，新的敌人会直接通过
- 阻挡数为0的干员（狙击、医疗、术师）不会阻挡敌人
- 合理利用阻挡数控制敌人流量

### 敌人类型
- 👹 小兵：基础敌人
- 😈 精英兵：更强属性
- 🛡️ 重甲兵：高防御
- 🦅 飞行兵：快速移动
- 👺 首领：BOSS级敌人

## 操作指南

### 基础操作
1. **购买干员**：点击商店中的干员卡片
2. **部署干员**：
   - 从备战区拖拽到战场空地
   - 或点击备战区干员，再点击战场空地
3. **撤回干员**：从战场拖拽回备战区
4. **卖出干员**：右键点击或Ctrl+点击备战区干员（获得1💰）
5. **查看攻击范围**：点击战场上的干员显示攻击范围（再次点击取消）

### 战斗流程
1. 准备阶段：部署干员、调整阵容
2. 点击"⚔️ 开始战斗"（战斗开始后无法操作）
3. 干员自动攻击范围内敌人
4. 敌人沿路径移动，被干员阻挡时攻击干员
5. **阻挡机制**：干员只能阻挡其阻挡数个敌人，超过阻挡数的敌人会直接通过
6. 击败所有敌人获胜，敌人到达保护点扣生命
7. **重要**：回合结束后，所有干员在原位置满血复活，恢复可操作状态

### 商店系统
- **刷新商店**：消耗2💰刷新5个干员
- **升级商店**：消耗5/10/15/20/25💰提升等级
- **商店等级**：等级越高，出现高星干员概率越大

### 合成系统
- 三个同名同精英度干员自动合成
- 精一干员：属性提升50%，标记★
- 精二干员：属性提升100%，标记★★

### 经济系统
- 初始金币：5💰
- 第2回合：3💰
- 第3回合起：每回合+1💰（3→4→5...最高10💰）
- 击杀敌人：+1💰
- 卖出干员：+1💰
- **重要**：金币不继承到下回合！

## 游戏提示

1. **合理部署**：重装放在路径上阻挡，远程放在后方输出
2. **阻挡策略**：注意干员阻挡数，避免敌人突破防线
3. **查看敌人**：点击"👁️ 查看敌人"了解下波敌情
4. **攻击范围**：点击干员查看攻击范围，优化站位
5. **经济管理**：每回合金币清零，合理花费
6. **合成策略**：收集同名干员合成精二获得强力单位
7. **商店升级**：适时升级商店获取高星干员
8. **💚 自动复活**：回合结束后所有干员满血复活，可以放心战斗！

## 快捷键
- **右键**：卖出备战区干员
- **Ctrl+点击**：卖出备战区干员
- **拖拽**：部署/撤回干员
